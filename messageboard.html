MESSAGE BOARD

WHY?

The reason I built this project was just to test my current knowledge of backend coding and setting up a small database, plus Iv always wanted to build my own persistent public message/chat board.

HOW?

  • Customer uploads picture.
  • 'Axio' hooks into 'Cloudinary' API, which runs a filter called cartoonify.
  • I receive the 'cartoonified' image back from 'Cloudinary' api, then run 'thresholding' on it (marvinJ library), this gets rid of any artifact grey colours, and leaves bold black and white, the neat thing is theres an option to pick a colour out and make it transparent, inside the threshold function.
  • So now I have a mask of the uploaded image, all I do now is set the background colour to what ever the customer picks, giveing them a good example of what thier sticker will look like.
  • There is also options to enter a name and pick a Font, which is all done with basic javascript/html.
